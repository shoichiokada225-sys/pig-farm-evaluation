# 分娩舎 従業員評価システム

養豚分娩舎で働く従業員を上司（管理者）が毎日評価するためのスマホ対応 Web アプリ。

## 対応言語

- 日本語 (JP)
- English (EN)
- Tiếng Việt (VI)
- Bahasa Indonesia (ID)

ヘッダー右上の言語ボタンで切替。選択した言語は localStorage に保存される。

## 動作確認手順

1. `index.html` をブラウザで開く
2. Chrome DevTools（F12）→ デバイスツールバー（Ctrl+Shift+M）でスマホ表示を確認

### 入力画面

1. 評価日・評価者名・被評価者名を入力
2. 10項目それぞれで 1〜5 のスコアボタンをタップ → 色が変わる
3. 任意でコメントを入力
4. 「保存」→ トースト「保存しました」
5. スコア未入力で保存 → 警告＋未入力項目にスクロール
6. 「リセット」→ 確認後クリア

### 編集機能

1. 履歴画面で評価をタップ → 詳細モーダル
2. 「編集」ボタン → 入力画面にデータが読み込まれる
3. 画面上部にオレンジの「編集中」バナー表示
4. 修正後「更新」ボタンで上書き保存
5. 「取消」ボタンで編集を中断

### 履歴画面

- 被評価者フィルター、タップで詳細、削除、CSV出力

### グラフ画面

- 被評価者選択 → 平均点折れ線グラフ ＋ レーダーチャート

## 技術仕様

- 1ファイル完結（`index.html`）
- 外部依存: Chart.js v4（CDN）
- データ保存: `localStorage`（キー: `pig_farm_evaluation_v1`）
- 言語設定: `localStorage`（キー: `pig_farm_eval_lang`）
- 自動保存: 300ms デバウンス
